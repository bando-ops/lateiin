Lösung

Das Perfekt kann im Lateinischen auf verschiedene Arten gebildet werden. Man erkennt die Art an der Veränderung des Verbstamms:

  • ascendi → Stammperfekt: Perfektstamm = Präsensstamm (ascend-).
  • servavi → v-Perfekt: Stamm + -v- (serva- + v + i).
  • scripsi → s-Perfekt: Stamm + -s-, oft mit Lautveränderung (scrib- + s → scrips-).
  • dedi → Reduplikationsperfekt: Anlaut des Stammes wird verdoppelt.
  • veni → Dehnungsperfekt: Stammvokal wird gedehnt (ven- → vēn-).
  • monui → u-Perfekt: Stamm + -u- (mone- + u + i).
  • potui → esse, posse, ire: unregelmäßige Verben mit eigenen Perfektformen.
